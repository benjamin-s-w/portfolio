Popcitive

This project was to produce a motion graphic logo that would serve as an animated web ad/bumper or YouTube/TV commercial. Using Adobe AfterEffects, I started out with the idea of someone getting their yearbook signed and commented on, then signified an update through visual transformation into some visual elements borrowed from the Drupal OpenSocial style guide.

Popcitive Open Source Social Media Commercial from Benjamin Woodruff on Vimeo.

Consumerism Design

Consumerism is a great concern for a lot of people, especially here in the Pacific Northwest, so I decided to do a visual design to help communicate how I feel this problem can be and has been addressed successfully in the past.
