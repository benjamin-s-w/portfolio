Visual Design

The intent of this project was to design an app for Tate art galleries that would be produced in tandem with the hypothetical opening of a new location scheduled to open soon in the USA. First, I created an app icon in the Assembly iPhone app. Next, I made a launch screen with the icon/logo tiled diagonally on a white background (also done in Assembly). Then, using Adobe Illustrator, I designed a landing screen with the USA location page content loaded up in a list with images (all scaled to the iPhone 5 screen size). Last, I animated the previous elements within an iPhone 5 mockup using Adobe AfterEffects to present the experience.
