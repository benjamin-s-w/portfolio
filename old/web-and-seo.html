Web Design / SEO

Below are some of Benjamin Woodruff's Web Design and SEO projects. Thanks for stopping by!

Enhanced Portfolio Site

I needed a responsive portfolio site that would be well organized yet visually pleasing. Part of my challenge was to make my site more easily recognizable to search engines, which prompted me to implement Schema.org markup, which uses Microdata to tell Google, et al. what my content is about.

Stuffed Hash Brown Omelette

As one of my favorite breakfasts, I felt it would be helpful for my wife to have a handy mobile-friendly website to access her recipe. It is responsive and features a simplified index card visual design.
